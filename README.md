# JanMat

# 🗳️ JANMAT - Empowering Citizens. Enhancing Governance.

JANMAT is a digital civic engagement platform that bridges the gap between citizens and governance. It enables users to voice concerns, provide feedback on governance, track ministers' activities, and express public opinions.

![image](https://github.com/user-attachments/assets/36336b2c-49f7-4897-9f65-ff49ed1e9bb1)

## 🌟 Key Features

### 👥 User Roles
- **Normal User**: Can submit complaints, participate in polls, and view project info.
- **Admin**: Manages the platform, users, content moderation, and analytics.
- **Official**: Responds to complaints, manages project details, and interacts with public feedback.

---

### 📌 Core Modules

- 🏛️ **Minister Info**  
  View profiles of elected leaders, their departments, and performance insights.

- 💰 **Projects & Funds**  
  Track development projects, fund allocations, and real-time updates from officials.

- 📝 **Complaints**  
  Citizens can raise issues related to governance and local administration.

- 📬 **Complaint Response**  
  Officials and authorities can reply with action taken reports and status updates.

- 📊 **Feedback Polls**  
  Allows users to participate in opinion polls on various topics, laws, or governance quality.

- 📣 **Public Opinion on Ministers**  
  Collects community sentiments and feedback on specific ministers and their work.

- 🔔 **Notifications**  
  Real-time updates on complaint statuses, polls, government actions, etc.

- ⚙️ **Settings**  
  Manage user preferences, profile settings, and role-based access.
